
# Food Delivery Data Analysis

## Objective
This project analyzes food delivery data by merging transactional,
user, and restaurant datasets to derive meaningful business insights.

## Datasets Used
- orders.csv
- users.json
- restaurants.sql

## Tools Used
- Python
- Pandas
- SQLite
- Matplotlib
- Google Colab

## Highlights
- Gold membership impact on revenue
- City-wise revenue analysis
- High-rated restaurant performance
